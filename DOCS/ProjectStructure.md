# 📂 Project Structure

## Overview

The Luxeon Perfume Store project follows a modular architecture that separates the user interface, styling, and application logic into dedicated files.

```
Luxeon-Perfume-Store
│
├── HTML Pages
├── CSS Stylesheets
├── JavaScript Modules
├── Images
├── README.md
└── LICENSE
```

### HTML
Contains the website pages.

### CSS
Contains all styling and responsive layouts.

### JavaScript
Contains application logic, shopping cart functionality, wishlist management, contact form handling, and Supabase integration.

### Database

Supabase stores:

- Contact Messages
- Shopping Cart
- Wishlist

This modular design makes the project easier to maintain and extend.
