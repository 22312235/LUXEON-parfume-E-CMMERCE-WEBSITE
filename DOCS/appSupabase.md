# ☁️ appSupabase.js

## Purpose

Acts as the application's database service layer.

## Responsibilities

- Creates user session IDs
- Stores wishlist items
- Removes wishlist items
- Loads wishlist data
- Stores shopping cart
- Loads shopping cart
- Sends contact messages

## Database Tables

- contact_messages
- wishlist_items
- cart_items
- cart_summary

This file centralizes all communication with Supabase.
