# 📖 About the Project

**LUXEON** is a modern luxury perfume shopping website that delivers a clean, elegant, and responsive online shopping experience.

The application allows users to browse premium fragrances, manage their shopping cart, save favorite products to a wishlist, and contact the store through an integrated contact form connected to **Supabase**.

> **📚 This project was developed for educational purposes as part of the Internet Programming I (CMPE233) course.**
